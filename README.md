# Proyecto-7
El proyecto tiene como finalidad crear un aplicación web que tenga un boton y una checkbox, el boton al ser presiionado creara un histograma y la checkbox un diagrama de dispersion.Se utilizara la informacion del dataset adjunto vehicles_us.
Utilizamos un repositorio de Github para almacenar el proyectoy el servicio será lanzado mediante la plataforma Render

El proyecto se puede consultar con el link:https://proyecto-7-j1ww.onrender.com/
